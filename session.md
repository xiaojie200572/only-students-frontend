## 1. 数据库表

```sql
-- 收藏夹表
CREATE TABLE favorite_folder (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    user_id BIGINT NOT NULL COMMENT '用户ID',
    name VARCHAR(50) NOT NULL COMMENT '收藏夹名称',
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    INDEX idx_user_id (user_id)
);

-- 修改收藏记录表，添加收藏夹ID
ALTER TABLE note_favorite ADD COLUMN folder_id BIGINT DEFAULT NULL;
```

## 2. API 接口

**获取收藏夹列表**：
```
GET /favorite/folders
Response: [{ id, name, count }]
```

**创建收藏夹**：
```
POST /favorite/folders
Body: { name }
Response: { id, name, count: 0 }
```

**获取指定收藏夹的收藏**：
```
GET /favorite/my?folderId=xxx
// folderId 为空时返回全部
```

**移动收藏到收藏夹**：
```
PUT /favorite/{favoriteId}/folder
Body: { folderId }
```
